# Security Policy

## Supported Versions

The following versions of `agent-runner` are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to security@metyatech.com. All security vulnerabilities will be promptly addressed.
